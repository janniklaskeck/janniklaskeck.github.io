Mayhem Heroes

PC | Third Person Shooter | Unreal Engine 4

Mayhem Heroes is a 4 player splitscreen third person shooter, where the players are trying to fight against the evil turned Candy King and his army of muffins.
Each player plays one of four classes(Healer, Mage, Mechanic and Scientist), each of which have 2 different abilities: The simpler primary attack, which charges up to a stronger version with each hit, and a strong special ability with a longer cooldown.
These abilities were designed to interact with each other(Example: The Blackhole ability of the scientist transforms to a electrified blackhole when hit by the chain lightning of the mage, now also dealing lightning damage).
Additionally an optional smartphone controller was developed which offered the same controls as a traditional gamepad for the game.

Livestreamer Twitch GUI

Desktop Application | Java (+ JavaFX)

Livestreamer Twitch GUI started as a GUI application for the livestreamer cli application, providing an easy way to manage favorite channels on numerous streaming sites in a simple GUI.
It's main purpose was managing favorite twitch.tv channels with special functionality, such as online status, preview images and a primitive chat client.
Other streaming services are also supported but only in a very generic way, without any of the the twitch.tv functionality.
It since then evolved to also support the more up-to-date Streamlink application and use the more modern GUI API JavaFX.
A twitch.tv games and channel browser was also added.
